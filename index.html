About this deck

This presentation is built with plain HTML, CSS, and a small amount of JavaScript — no build tools, no external dependencies. Open index.html in any modern browser to get started.

Use the arrow keys or the navigation buttons at the bottom to move between slides. Edit the <section> elements in the source to add your own content.

How to Customise

  • Add a slide — copy any <section class="slide …"> block and change the content.
  • Change colours — edit the :root CSS variables at the top of the <style> block.
  • Change the font — replace the font-family on body with any web-safe or Google Font.
  • Add images — drop an <img> into any slide; they scale automatically.
  • Export to PDF — open in Chrome, press Ctrl+P, choose "Save as PDF".

Embedding Code

Use the .code-wrap block to highlight a snippet:

// Greet the audience
