# Public Financial Management (PFM) Act Presentation

This repository branch contains a ready-to-use slide deck outline for a 12–15 minute group presentation on the **Public Financial Management (PFM) Act**.

## Files
- `PFM_Act_Presentation.md` - Full slide-by-slide presentation content with speaker notes and references.

## Presentation summary
- Recommended group size: **3–5 students**
- Recommended duration: **12–15 minutes**
- Number of slides: **13 slides**

## Core topics covered
- Background and objectives of the PFM Act
- Key provisions
- Roles and responsibilities under the Act
- Importance for public sector accountability and transparency
- Critical analysis
- Real-world examples / case studies
- Conclusion and recommendations
- References

## Suggested use
You can copy the slide content into PowerPoint, Google Slides, Canva, or any presentation tool and divide speaking roles among group members.
